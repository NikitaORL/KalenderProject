📅 Uue kalendri loomine Microsoft Projectis

Samm-sammult juhend koos seletuste ja ekraanipiltidega

Loo kalender Tööaja muutmine Kalendri kasutamine
Formula

1. Uue kalendri loomine

  1. Ava Microsoft Project.
  2. Vali Project → Change Working Time.
  3. Klõpsa Create New Calendar.
  4. Anna kalendrile nimi

2. Tööaja muutmine

  1. Vali kalender ja klõpsa Details.
  2. Määra tööajad nädalapäevade kaupa.
  3. Lisa eripäevad

3. Kalendri kasutamine

  1. Topeltklõps ülesandel.
  2. Vali kalender
  3. Projekt arvestab tööaega
  4. Tööaja kontroll

Kalendri mõju ajakavale

  • Töö- ja puhkepäevad
  • Tööaeg
  • Erandid
