📅 Uue kalendri loomine Microsoft Projectis

Samm-sammult juhend koos selgituste ja ekraanipiltidega

Kalender Formula Diagrammid

1. Uue kalendri loomine

  1. Avage Microsoft Project.
  2. Valige Project → Change Working Time (Projekt → Muuda tööaega).
  3. Klõpsake Create New Calendar (Loo uus kalender).
  4. Andke kalendrile nimi (näiteks "Minu töökalender").

2. Tööaja muutmine

  1. Valige loodud kalender ja klõpsake Details (Üksikasjad).
  2. Määrake tööaeg iga nädalapäeva jaoks (näiteks E-R 9:00-17:00).
  3. Lisage eripäevad (puhkepäevad, pühad, lühendatud päevad).

3. Kalendri kasutamine

  1. Topeltklõpsake ülesandel, millele soovite kalendrit rakendada.
  2. Valige loodud kalender rippmenüüst.
  3. Project arvestab määratud tööaega.
  4. Kontrollige tööaega kalendrivaates.

Kalendri mõju ajakavale

  • Töö- ja puhkepäevad mõjutavad ülesannete kestust
  • Tööaeg määrab ressursside kättesaadavuse
  • Erandid (pühad) lükkavad tähtaegu edasi
