📅 Uue kalendri loomine Microsoft Projectis

Samm-sammult juhend koos seletuste ja ekraanipiltidega

1. Uue kalendri loomine

Kalendri loomiseks:

  1. Ava Microsoft Project.
  2. Vali Project → Change Working Time.
  3. Klõpsa Create New Calendar.
  4. Anna kalendrile nimi

2. Tööaja muutmine

Kalendri tööaja muutmiseks:

  1. Vali kalender ja klõpsa Details.
  2. Määra tööajad nädalapäevade kaupa.
  3. Lisa eripäevad (näiteks pühad)

3. Kalendri kasutamine

Kalendri rakendamiseks ülesandele:

  1. Topeltklõps ülesandel.
  2. Väljal Calendar vali loodud kalender.
  3. Projekt võtab arvesse töö- ja puhkepäevi.
  4. Tööaeg oli järgmine:

    5. Kalendri mõju ajakavale

    Uus kalender mõjutab otseselt ülesannete tähtaegu ja projekti graafikut. See arvestab:

    • Töö- ja puhkepäevi (näiteks nädalavahetused ja pühad).
    • Tööaega iga päeva kohta (nt 8:00–17:00).
    • Erandeid, näiteks puhkused või ärireisid.

    Kalendri muutmisel arvutab Project automaatselt ülesannete algus- ja lõppkuupäevad, uuendab Gantti diagramme ja hoiab projekti ajakava täpse.
