📅 Uue kalendri loomine Microsoft Projectis

Samm-sammult juhend koos selgituste ja ekraanipiltidega

Kalender Formula Diagrammid

1. Uue kalendri loomine

  1. Avage ProjectLibre.
  2. Valige File → Calendar (Projekt → Muuda tööaega).
  3. Klõpsake Create New Calendar (Loo uus kalender).
  4. Andke kalendrile nimi (näiteks "Minu töökalender").

2. Tööaja muutmine

  1. Määrake tööaeg iga nädalapäeva jaoks (näiteks E–R 9:00–17:00).
  1. Kui soovite midagi muuta, kasutage Duration seadeid (näiteks tööpäeva kestus).

3. Kalendri kasutamine

  1. Kalendri rakendamiseks topeltklõpsake ajaväljal oma kalendris.
  1. Seejärel minge menüüsse: Advanced → Task Calendar ja valige oma kalender.
  1. Project arvestab määratud tööaega.

Kalendri mõju ajakavale

  • Töö- ja puhkepäevad mõjutavad ülesannete kestust.
  • Tööaeg määrab ressursside kättesaadavuse.
  • Erandid (pühad) võivad tähtaegu edasi lükata.
