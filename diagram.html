📊 Diagrammid ProjectLibres

Õpi, kuidas luua diagramme ja jälgida ülesandeid ning ressursse projektis.

Kalender Diagrammid

Diagrammitüübid ProjectLibres

  • Gantt Chart (Gantti diagramm) – näitab ülesannete kestust ja nende sõltuvusi ajaskaalal.
  • Network Diagram (Võrgustiku diagramm) – kuvab ülesandeid ja nende järjekorda nooledega, kasulik kriitilise tee jälgimiseks.
  • Resource Usage / Resource Sheet (Ressursside tabelid) – tabelid, mis näitavad ressursside (inimesed, seadmed) jaotust. Otse graafikuid ei kuvata.

1. Gantt Chart loomine

  1. Avage ProjectLibre ja looge uus projekt (File → New).
  2. Lisage ülesanded: Task Name ja Duration (näiteks 5d).
  1. Määrake sõltuvused: Predecessors veerg või Link Tasks nupp.
  2. Diagramm kuvatakse automaatselt paremal (Gantt Chart).

2. Network Diagram loomine

  1. Valige menüüst View → Network Diagram.
  2. Diagrammil kuvatakse ülesanded plokkidena ja sõltuvused nooledega.
  3. Kasutatakse ülesannete järjestuse ja kriitilise tee jälgimiseks.

3. Ressursside jälgimine

  1. Menüü View → Resource Sheet – ressursside tabel.
  2. Menüü View → Resource Usage – tabel, kus näete, milliseid ressursse ülesanded kasutavad.
  3. Otse graafikuid ProjectLibre ei loo; graafikud saab teha, eksportides andmed Excelisse.

Kokkuvõte

ProjectLibre võimaldab visualiseerida ülesannete ajaraame (Gantt Chart) ja ülesannete vahelisi seoseid (Network Diagram), samuti jälgida ressursside tabelit. Graafikute ja diagrammide loomiseks, nagu Excelis, tuleb andmed eksportida.
